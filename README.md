## About
This repository contains a simple chatbot application designed to explore the capabilities of generative AI. The primary objective is to gain hands-on experience in building an application that consumes Googleâ€™s Gemini AI.

## Installation
1. Get a API Key at [Google AI Studio](https://aistudio.google.com/app/apikey)

2. Clone the repo
    ```bash
    https://github.com/alexandrecorazza/a-chatbot-app.git
    ```

3. Install NPM packages
    ```bash
    npm install
    ```

4. Enter your API Key in `.env.local`
    ```bash
    GOOGLE_GENERATIVE_AI_API_KEY = 'YOUR API KEY';
    ```

## Running it
```bash
npm run dev
```
